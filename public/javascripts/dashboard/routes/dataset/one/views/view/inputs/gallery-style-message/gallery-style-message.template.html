Your gallery can have different styles. The Image Gallery is the default, and will display either text only or images specified in your data source. The Icon Gallery will let you choose images to represent specific categories of data that repeat for each tile, for example images for Male and Female. Finally, the Heatmap Gallery lets you display tiles with color backgrounds, mapped to fields in your data. Color mappings are defined in General Settings.
